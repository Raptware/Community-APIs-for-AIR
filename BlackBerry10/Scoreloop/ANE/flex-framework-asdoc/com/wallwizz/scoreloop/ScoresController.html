loadScores()method 
public function loadScores(range:Range):void

This method requests the list of scores, defined by the supplied range, from the server. Note that this is an asynchronous call and a callback will be triggered, after which you can access the retrieved scores by calling getScores().

Parameters

range:Range — The starting position and offset of the requested scores range (numbered from 0, the first score to load).

loadScoresAroundUser()method 
public function loadScoresAroundUser(length:int):void

This method returns the list of scores that includes the given score in the middle. Note that this is an asynchronous call and a callback will be triggered, after which you can access the retrieved scores by calling getScores().

Parameters

length:int — The length of the range requested.

loadScoresAtRank()method 
public function loadScoresAtRank(rank:int, length:int):void

Requests the list beginning at the given rank. The rank refers to a score's position on the global leaderboard. All lists start at 1, therefore the minimum rank possible is 1. Note that this is an asynchronous call and a callback will be triggered, after which you can access the retrieved scores by calling getScores().

Parameters

rank:int — The starting position of the range (numbered from 1, the first score to load).
 
length:int — The length of the range requested.
