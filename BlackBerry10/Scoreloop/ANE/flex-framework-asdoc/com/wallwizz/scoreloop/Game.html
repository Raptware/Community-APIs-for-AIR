Classpublic class Game
InheritanceGame Object

The Game class models the game application. Games must be registered at https://developer.scoreloop.com, where they are assigned a unique game-id and game secret, and where you can configure additional game properties. Game provides access to the game identifier and game secret that were assigned by Scoreloop when you registered the game.
